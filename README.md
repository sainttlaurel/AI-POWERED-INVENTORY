# Inventory Management System

A modern inventory management system built with **PHP, MySQL, and Bootstrap** featuring AI-powered forecasting, profit tracking, and a smart chatbot assistant.

---

## Features
- Product & inventory management
- Profit tracking with margin calculations
- AI-based stock forecasting
- Reservation system with booking codes
- Analytics and reporting dashboard
- AI chatbot for inventory queries
- Glassmorphism UI with responsive design

---

## Tech Stack
- PHP
- MySQL / MariaDB
- Bootstrap
- JavaScript

---

## Setup

- git clone: https://github.com/sainttlaurel/AI-POWERED-INVENTORY.git

- Move the project to your `htdocs` folder
- Run `php setup_advanced_forecasting.php`
- Open `http://localhost/inventory-system/`
