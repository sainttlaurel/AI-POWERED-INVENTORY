# supabase setup

sql files for cloud database.

## files

- `001_create_core_schema.sql` - creates tables
- `002_enable_rls_policies.sql` - security policies

## how to run

1. go to https://supabase.com/dashboard/project/temdnfcnllnvpgpookex/editor
2. copy paste `001_create_core_schema.sql`
3. run it
4. copy paste `002_enable_rls_policies.sql`
5. run it

done.

## what it creates

16 tables: users, products, categories, suppliers, sales, invoices, inventory_logs, notifications, forecast_data, etc.
